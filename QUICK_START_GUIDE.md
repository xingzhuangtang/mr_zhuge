# 🎮 快速启动指南 - 游戏化军事历史可视化系统

## 🚀 启动系统

### 第一步：清理端口（如果需要）
```bash
# 如果遇到端口占用错误，先清理端口
lsof -ti:8000 | xargs kill -9
```

### 第二步：启动系统
```bash
./start_game_battle_branch.sh
```

### 第三步：访问系统
在浏览器中访问：
- **主入口页面**: http://localhost:8000/static/index.html
- **游戏化系统**: http://localhost:8000/static/game-battle-replay.html

## 🎯 测试AI聊天功能

### 1. 进入游戏化系统
访问 http://localhost:8000/static/game-battle-replay.html

### 2. 选择战役
点击选择"🏛️ 赤壁之战 (208年)"

### 3. 开启AI助手
点击右下角的蓝色💬聊天按钮

### 4. 测试对话
在聊天框中输入以下问题测试：

**智能上下文测试**：
- "这个战役的主要战术是什么？"
- "曹操的军队有什么优势？"
- "火攻是如何进行的？"

**一般军事历史问题**：
- "重装步兵方阵有什么特点？"
- "骑兵在古代战争中的作用？"
- "火攻战术在军事史上的重要性？"

### 5. 观察AI回答
AI应该能够：
- 识别当前观看的赤壁之战
- 提供专业的军事分析
- 结合战役背景进行讲解

## 🔧 故障排除

### 问题1：端口占用
```bash
# 解决方案
lsof -ti:8000 | xargs kill -9
```

### 问题2：AI聊天不工作
1. 检查浏览器控制台是否有错误
2. 确认AI助手面板已开启
3. 检查网络连接

### 问题3：3D地图不加载
1. 刷新页面 (F5)
2. 检查网络连接
3. 查看浏览器控制台错误

### 问题4：战役选择无响应
1. 确认战役数据加载完成
2. 检查JavaScript控制台错误
3. 重新加载页面
